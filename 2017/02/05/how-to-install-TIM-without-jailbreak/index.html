iOS设备不越狱安装TIM

Posted by Terence Han on 2017-02-05

腾讯TIM和之前的TM2013、QQ轻聊版、国际版等似乎有相似之处，但又有本质不同。TIM不仅是给普通QQ瘦身，更是增加了“肌肉”，比如协作文档、邮件和日程等，这既做到了轻聊，又能够为提高工作效率添砖加瓦，让用户在上班的时候也能“任性聊QQ”。

安装之前

安装TIM之前需要用电脑下载如下内容，并将iOS设备链接到电脑（需要有iTunes环境）

  • 腾讯TIM 链接
  • Cydia impactor 链接 密码:vccp

安装教程

  • 解压并运行Cydia impactor
  • 拖动下载好的ipa到Cydia impactor里

  • 输入你的AppleID和密码，用于申请临时证书。若AppleID开启了二次验证，到Apple官网生成app专用密码进行登陆


  • 安装过程中会弹出一个警告，点击OK，之后Cydia impactor会先对此App进行签名，然后会自动将签名后的App安装到设备中
  • 然后在手机中一次点击设置-通用-描述文件与设备管理。可以看到开发者应用下有一个你的AppleID，点击并进入，信任应用即可。
